# ⚽ בית"ר ירושלים - לומדים מספרים 1-20

אפליקציית ווב אינטראקטיבית ללימוד מספרים בעברית לילדים, עם עיצוב בצבעי בית"ר ירושלים (שחור וצהוב) ונושא כדורגל.

## 🚀 איך להריץ את האפליקציה

### הרצה מקומית
1. פתחו את הקובץ `index.html` בדפדפן
2. זהו! האפליקציה עובדת ללא צורך בהתקנה או build

### פרסום ב-GitHub Pages
1. היכנסו ל-**Settings** → **Pages** בריפו
2. תחת **Source**, בחרו **Deploy from a branch**
3. בחרו **Branch: main** ו-**Folder: / (root)**
4. לחצו **Save** וחכו 1-2 דקות
5. האפליקציה תהיה זמינה בכתובת: `https://[USERNAME].github.io/[REPO-NAME]`

## ✨ מה יש באפליקציה

### מצב למידה 📚
- תצוגה גדולה של מספר והמילה בעברית
- הצגה ויזואלית של כדורי כדורגל (כמספר הכדורים)
- כפתורי ניווט: הקודם, הבא, אקראי
- כפתור "הקרא" עם דיבור בעברית (Speech Synthesis)

### מצב חידון 🎯
שלושה סוגי שאלות שונים:
- **בחר את המספר**: מילה בעברית → בחר את הספרה
- **בחר את המילה**: ספרה → בחר את המילה בעברית
- **ספור כדורים**: כדורי כדורגל → בחר את המספר

מאפיינים:
- מערכת ניקוד ורצפים
- שמירה אוטומטית ב-localStorage
- משוב מיידי על תשובות
- כפתור "הקרא" לשאלה והתשובה

### מצב סדר נכון 🎴
- 20 קלפים עם מספרים ומילים בעברית
- גרירה ושחרור (drag & drop) לסידור
- כפתור "בדוק" לאימות הסדר
- כפתור "ערבב מחדש" להתחלה חדשה
- +50 נקודות על הצלחה

## 🎨 עיצוב
- רקע שחור עם צהוב דומיננטי (צבעי בית"ר ירושלים)
- תמיכה מלאה ב-RTL (עברית)
- מותאם למובייל וטאבלט
- ללא תלות חיצונית - HTML, CSS ו-JavaScript טהורים

## 🛠️ טכנולוגיות

- CSS3 (עם Flexbox ו-Grid)
- JavaScript Vanilla (ES6+)

- LocalStorage API (שמירת ניקוד)
- Drag and Drop API
