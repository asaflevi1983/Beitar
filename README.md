# בית&quot;ר ירושלים - לומדים מספרים עד 20

אפליקציית ווב קטנה (GitHub Pages) ללימוד מספרים 1–20 לילדים, בנושא כדורגל ובצבעי בית&quot;ר ירושלים (שחור/צהוב).

## איך מריצים
1. היכנסו ל-Settings → Pages
2. בחרו Deploy from a branch
3. Branch: `main` / Folder: `/ (root)`
4. שמרו וחכו דקה-שתיים

ואז האפליקציה תופיע בכתובת GitHub Pages של הריפו.

## מה יש באפליקציה
- מצב **לימוד**: מספר גדול + המילה בעברית
- מצב **חידון**: שאלות אמריקאיות + ניקוד ורצף
- שמירה מקומית בדפדפן (localStorage)
