# ⚽ בית"ר ירושלים - לומדים מספרים 1-30

אפליקציית ווב אינטראקטיבית ללימוד מספרים ואותיות בעברית לילדים, עם עיצוב בצבעי בית"ר ירושלים (שחור וצהוב) ונושא כדורגל.

## 🔗 קישור לאפליקציה
**[לחצו כאן להפעלת האפליקציה](https://asaflevi1983.github.io/Beitar/)** 🚀

## 🚀 איך להריץ את האפליקציה

### הרצה מקומית
1. פתחו את הקובץ `index.html` בדפדפן
2. זהו! האפליקציה עובדת ללא צורך בהתקנה או build

### פרסום ב-GitHub Pages
1. היכנסו ל-**Settings** → **Pages** בריפו
2. תחת **Source**, בחרו **Deploy from a branch**
3. בחרו **Branch: main** ו-**Folder: / (root)**
4. לחצו **Save** וחכו 1-2 דקות
5. האפליקציה תהיה זמינה בכתובת: `https://[USERNAME].github.io/[REPO-NAME]`

## ✨ מה יש באפליקציה

### מצב למידה 📚
- תצוגה גדולה של מספר והמילה בעברית (1-30)
- הצגה ויזואלית של כדורי כדורגל (כמספר הכדורים)
- כפתורי ניווט: הקודם, הבא, אקראי
- כפתור "הקרא" עם דיבור בעברית (Speech Synthesis)

### מצב חידון 🎯
- **ספור כדורים**: כדורי כדורגל → בחר את המספר (1-30)

מאפיינים:
- מערכת ניקוד ורצפים
- שמירה אוטומטית ב-localStorage
- משוב מיידי על תשובות
- כפתור "הקרא" לשאלה

### מצב סדר נכון 🎴
- 30 קלפים עם מספרים
- **לחיצה לבחירה** במקום גרירה ושחרור
- כפתור "אשר" לאימות הסדר
- כפתור "דחה" לביטול הבחירה
- כפתור "ערבב מחדש" להתחלה חדשה
- +50 נקודות על הצלחה

### מצב זיכרון 🧠
- משחק זיכרון קלאסי עם מספרים
- **שלושה גדלים**:
  - 4×4 (8 זוגות) - קל
  - 6×6 (18 זוגות) - בינוני
  - 8×8 (32 זוגות) - מתקדם
- מעקב אחר מהלכים וזוגות שנמצאו
- +100 נקודות על השלמת משחק

### מצב אותיות 🔤
- למידת האלף-בית העברי (27 אותיות)
- תצוגה גדולה של אות ושמה
- כפתורי ניווט: הקודמת, הבאה, אקראי
- כפתור "הקרא" עם דיבור בעברית
